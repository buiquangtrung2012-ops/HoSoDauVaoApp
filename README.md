# WordHoSo Desktop App

Dự án này sẽ được chuyển thành phần mềm desktop chạy trên Windows sử dụng Electron. Mã nguồn HTML/JS hiện tại sẽ được đóng gói lại để chạy độc lập.

## Yêu cầu
- Node.js
- Electron

## Hướng dẫn cài đặt và chạy
1. Cài Node.js (nếu chưa có)
2. Cài Electron: `npm install`
3. Chạy phần mềm: `npm start`

- Mã nguồn gốc: Add-in Word
- Đã xác nhận mục tiêu: Phần mềm desktop Windows
